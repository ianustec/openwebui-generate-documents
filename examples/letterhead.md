# Scope of work

This example shows how to reuse a **company letterhead** attached in the chat.

1. Attach `Carta_Rossi.docx` (or your own file) to the message.
2. Set `letterhead:` to that exact filename (or the stem without `.docx`).
3. Write the body in Markdown as usual.

::: callout type="info" title="Note"
Header, footer and page margins come from the attached file. Only this body
is generated by the tool.
:::
