# Assets

Screenshots used by the README and the announcement post. All are **100%
generated by the tool** from [`examples/report.md`](../examples/report.md).

- `cover.png` — band cover page
- `content.png` — numbered headings, callout, KPI table, bullets
- `callouts.png` — alert callout + native "layers" table
- `table.png` — risk matrix, danger callout, quote, signature
- `hero.png` — 3-page preview banner (used as the README header and post image)

Regenerate them from the example report:

```bash
python examples/build.py                             # -> examples/demo_report.docx
soffice --headless --convert-to pdf examples/demo_report.docx
pdftoppm -png -r 150 demo_report.pdf page             # one PNG per page
```
